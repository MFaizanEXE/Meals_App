# Meals_App
🍽️ Meals App: Flutter Recipe & Meal Planner

Meals App is a feature-rich Flutter application designed to help users explore, filter, and manage different meals based on their preferences. With an interactive UI and smooth navigation, users can browse recipes, apply dietary filters, and mark their favorite meals.
This project demonstrates the use of Flutter for building dynamic, multi-screen applications with efficient state management and structured navigation.

🚀 Project Objective

To build a multi-screen Flutter application
To implement navigation and routing between screens
To apply filtering logic based on user preferences
To manage and persist user-selected favorite meals

🛠 Core Concepts Used

Flutter Navigation: Implemented stack-based navigation between screens
State Management: Managed filters and favorite meals dynamically
Dart OOP: Structured data models for meals and categories
UI Design: Built responsive and visually appealing layouts

📦 Project Structure

Meals App/
│
├── main.dart          # Entry point and route configuration
├── models/            # Meal and category data models
├── screens/           # Different app screens (categories, meals, details)
├── widgets/           # Reusable UI components
└── README.md          # Documentation

🎯 Key Features

Feature	Description
Categories	Browse meals by category
Meal Details	View ingredients and steps
Filters	Apply dietary filters (e.g., gluten-free, vegan)
Favorites	Save and manage favorite meals
Smooth Navigation	Seamless screen transitions
